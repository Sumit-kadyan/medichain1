# **App Name**: MediChain

## Core Features:

- Role-Based Dashboards: Role-based dashboards for Receptionist, Doctor, and Pharmacy with clear task separation and data views, isolated to the current user's clinic ID.
- Patient Flow Management: Patient Flow Management enabling receptionist actions: Add Patient, Continue Patient, Renew Patient with receipt and visit generation, following a validity-based system.
- Receipt Template Editor: HTML-based Receipt Template Editor allowing customization with live preview, supporting placeholders such as clinic and patient details.
- PDF Generation: PDF Generation for prescriptions, pharmacy bills and receipts
- Patient History Timeline: Patient History provides a summarized timeline of visits, prescriptions, and receipts.
- AI Drug Suggestions: AI tool powered smart suggestion engine for drug prescriptions based on patient history and chief complaints, with human oversight and final approval remaining with the doctor.

## Style Guidelines:

- Primary color: A soft, professional blue (#6699CC) evoking trust and reliability. Blue is commonly associated with healthcare and professionalism.
- Background color: A very light, desaturated blue (#F0F8FF), offering a clean and calm interface.
- Accent color: A muted green (#8FBC8F), complementary to the primary blue, used for positive actions and highlights.
- Font pairing: 'Inter' (sans-serif) for body text and 'PT Sans' (sans-serif) for headlines, for a clean and modern look, and good readability.
- Code font: 'Source Code Pro' for any displayed code, providing clear distinction and readability.
- Consistent use of simple, clear icons to represent actions and categories throughout the app.
- Clean and intuitive layout with clear sections for patient information, waiting lists, and management tools.
